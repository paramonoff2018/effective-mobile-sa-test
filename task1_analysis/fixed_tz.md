# Исправленная версия ТЗ «Корзина»

## 1. Общие ограничения
- Максимальное количество различных товаров в корзине — 5.
- Максимальное общее количество всех товаров (сумма единиц) — 20.
- Минимальное количество одного товара — 1, максимальное — 10.

- ## 2. Добавление товара в корзину
- При добавлении товара в корзину система проверяет соблюдение всех лимитов:
  - если товар уже есть в корзине, проверяется, не превысит ли новое количество лимит в 10 единиц для этого товара;
  - если товара нет в корзине, проверяется, не будет ли превышен лимит в 5 различных товаров;
  - в любом случае проверяется, не превысит ли общее количество всех товаров 20.
- Также проверяется доступное количество товара на складе:
  - если остаток меньше желаемого, пользователь получает сообщение: «Доступно только X единиц товара»;
  - добавить можно не больше, чем есть в наличии.
- В зависимости от того, какой лимит или ограничение нарушается, показывается одно из сообщений:
  - «Вы можете добавить не более 10 единиц этого товара»;
  - «В корзине не может быть больше 5 разных товаров»;
  - «Общее количество товаров в корзине не может превышать 20»;
  - «Доступно только X единиц товара» (где X — остаток на складе).

## 3. Изменение количества товаров и удаление
- Пользователь может изменить количество любого товара в корзине в пределах от 0 до 10 (или до остатка на складе, если остаток меньше 10).
- При установке количества 0 товар автоматически удаляется из корзины.
- Предусмотрена отдельная кнопка «Удалить», которая выполняет то же действие — устанавливает количество 0 и удаляет товар.
- После удаления освобождаются лимиты (на количество разных товаров и на общее число единиц).

## 4. Цена товаров в корзине
- Цена товара фиксируется в момент добавления в корзину и не меняется автоматически при изменении цены в каталоге.
- Если цена в каталоге изменилась, при следующем посещении страницы корзины пользователь видит предупреждение:
  - «Цена на товар [название] изменилась. Текущая цена — [новая цена]. Обновить цену в корзине?»
- Пользователь может согласиться на новую цену (нажав «Обновить») или оставить старую (нажав «Оставить старую»).
- Если пользователь обновляет цену, она пересчитывается для всех единиц этого товара.

## 5. Отображение корзины
- На странице корзины отображается:
  - список товаров с названиями;
  - количество каждого товара;
  - доступное количество на складе (если оно меньше запрошенного, рядом появляется предупреждение, например: «Осталось только X шт.»);
  - цена за единицу товара;
  - общая стоимость позиции (количество × цена);
  - итоговая сумма всего заказа.
- Если товар полностью отсутствует на складе, он помечается как «Нет в наличии» и автоматически блокируется для оформления заказа. Пользователь может удалить такой товар.

## 6. Реклама в корзине
- В корзине предусмотрено место для отображения рекламных блоков (например, товарных рекомендаций).
- Контент, время показа и частота рекламы настраиваются через административную панель и не влияют на лимиты корзины.
- По умолчанию реклама может показываться, но это не является обязательным требованием к функционалу корзины (реализуется как отдельный модуль).

## 7. Обработка ошибок и исключений
- Все ошибки, связанные с превышением лимитов или недостатком товара, сопровождаются понятными сообщениями (см. п.2).
- Если товар, добавленный в корзину, становится недоступен к заказу (например, закончился на складе), он отображается с пометкой «Нет в наличии» и блокируется для оформления заказа. Пользователь может удалить такой товар из корзины.
